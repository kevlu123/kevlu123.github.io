Notes

(Updated 14/06/21)
The menu displayed may not be accurate when there are holidays that week.
To see the true menu, press Ctrl+S to save a .pdf of the menu somewhere.
Do NOT notify me if the menu is incorrect.

Installation

Extract the .zip file.
WhatsForLunch is portable so you can save it can run it from anywhere on your computer without installation.

Run on Startup

To add the program to startup, first place the folder somewhere permanent.
After the folder is in a permanent place, right click WhatsForLunch.exe and create a shortcut.
Right click the shortcut and click properties.
Append /startup to the end of Target and press OK.
In File Explorer, navigate to the startup folder and move the shortcut here.
